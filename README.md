# ApexPlanet Internship - Task 3

## Task Name
Task 3 - Advanced Styling and JavaScript

## Project Description

This project was developed as part of the ApexPlanet Web Development Internship. It demonstrates advanced front-end development concepts by integrating multiple interactive JavaScript applications into a single responsive dashboard.

The project includes an Image Carousel, Quiz Application, and Weather Dashboard with a modern responsive interface.

## Objective

- Build advanced interactive web applications.
- Apply modern CSS styling techniques.
- Use JavaScript for dynamic user interactions.
- Create responsive layouts using Flexbox and CSS Grid.
- Improve user experience with animations and transitions.

## Features

### Image Carousel
- Automatic slideshow
- Previous and Next navigation
- Play and Pause functionality
- Navigation dots
- Smooth transitions

### Quiz Application
- Multiple-choice questions
- Progress indicator
- Instant answer validation
- Score calculation
- Restart quiz option

### Weather Dashboard
- City selection
- Dynamic weather information
- Loading animation
- Error handling
- Weather details display

### User Interface
- Responsive design
- Glassmorphism layout
- CSS Grid and Flexbox
- Smooth animations
- Modern color theme

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
